# Backend API Review

Review the requested backend area.

Check:

- Route/controller/endpoint
- Request DTO
- Response DTO
- Validation
- Authentication
- Authorization
- Ownership checks
- Tenant isolation
- Service logic
- Repository/query logic
- Error handling
- Logging
- Tests
- API compatibility

Output:

1. Endpoint map
2. Current flow
3. Security findings by severity
4. Correctness issues
5. Missing tests
6. Recommended fixes
7. Exact files involved

Do not edit files unless explicitly asked.
