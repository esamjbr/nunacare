# Pull Request Review

Review the current diff for:

- Correctness
- Security
- Tests
- Breaking changes
- Database impact
- Performance
- Maintainability
- Naming
- Dead code
- Missing docs

Output:

1. Summary
2. Must fix
3. Should fix
4. Nice to have
5. Questions for author
6. Test recommendations
7. Approval recommendation

Do not modify files unless explicitly asked.
