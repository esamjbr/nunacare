# Test Generation

Generate or recommend tests for:

- Happy path
- Invalid input
- Unauthorized access
- Forbidden access
- Not found
- Ownership mismatch
- Tenant mismatch
- Edge cases
- Regression cases

Rules:

- Follow existing test framework and style.
- Do not invent new testing infrastructure unless necessary.
- Prefer small, meaningful tests.
- Explain what each test proves.
